Lesson Deep Dive

Lesson 25 — Two Population Proportions

Objectives (SLOs 1, 5, 6):

  • Test \(p_1 - p_2\) with two independent samples
  • Construct and interpret CIs for \(p_1 - p_2\)
  • Verify conditions for the two-proportion z-test
TipKey Concepts
  • Pooled proportion: \(\hat{p} = \frac{x_1 + x_2}{n_1 + n_2}\) — used in the test statistic under \(H_0: p_1 = p_2\)
  • Test statistic: \(z = \frac{(\hat{p}_1 - \hat{p}_2) - 0}{\sqrt{\hat{p}(1-\hat{p})(1/n_1 + 1/n_2)}}\)
  • CI: \((\hat{p}_1 - \hat{p}_2) \pm z_{\alpha/2} \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}\)
  • Conditions: Both samples random and independent; \(n_1\hat{p}_1 \geq 10\), \(n_1(1-\hat{p}_1) \geq 10\), and same for sample 2
ImportantGrading Focus
  • Pooled vs unpooled SE: The test uses the pooled proportion \(\hat{p}\) in the SE (because under \(H_0\), \(p_1 = p_2\)). The CI uses the individual \(\hat{p}_1\) and \(\hat{p}_2\) (because we’re not assuming equality). This parallel mirrors the one-proportion case (\(p_0\) in test, \(\hat{p}\) in CI).
  • Conditions for both groups: Cadets must check the success/failure condition for each sample separately. Checking only one group is incomplete.
  • Interpretation: “We are 95% confident the true difference in proportions (\(p_1 - p_2\)) is between…” Include context about what the groups are.
NoteDeeper Material

Why pool under \(H_0\): If \(H_0: p_1 = p_2\) is true, then both samples come from populations with the same proportion \(p\). The best estimate of this common \(p\) is the pooled proportion. Using individual \(\hat{p}_1\) and \(\hat{p}_2\) in the test would be inconsistent with the null hypothesis.

Relative risk and odds ratios: In epidemiology and medical research, the difference \(p_1 - p_2\) is often less useful than the relative risk (\(\hat{p}_1/\hat{p}_2\)) or odds ratio. These aren’t covered in MA206 but are standard in applied statistics. If a cadet asks “is there a better way to compare proportions?” — yes, depending on context.

Fisher’s exact test: When sample sizes are small and the normal approximation fails, Fisher’s exact test computes exact p-values using the hypergeometric distribution. R’s fisher.test() does this. Worth knowing as a faculty member even though cadets use the z-test.

Lesson 26 — WPR II Review

Objectives (SLOs 1, 3, 5, 6):

  • Review and synthesize all Block II material (Lessons 17–25)
  • Practice test selection and complete hypothesis testing procedures
ImportantReview Focus

Prioritize these high-error topics:

  1. Test selection: Given a scenario, which test? This is the #1 skill for WPR II. Use a decision tree:
    • One sample → mean (\(t\)) or proportion (\(z\))?
    • Two samples → independent or paired? Means or proportions?
  2. Correct standard error: Each test has a different SE formula. The most common errors:
    • Using \(\sigma\) instead of \(s\) (or \(s/\sqrt{n}\))
    • Forgetting to pool proportions in a two-proportion test
    • Using \(2n - 2\) df for paired when it should be \(n - 1\)
  3. Conclusion language: “At \(\alpha = 0.05\), there is [sufficient/insufficient] evidence to conclude that…”
  4. Conditions: Every test requires conditions. Skipping them = lost points.
  5. CI-HT duality: If 0 is in the CI for a difference, we fail to reject. If \(\mu_0\) is outside the CI, we reject.

Grading calibration: Before grading begins, all instructors should grade the same 2–3 exams independently, then compare. Resolve rubric interpretation differences before grading the full set.

Common rubric issues from past semesters:

  • How much credit for correct test selection but wrong execution?
  • How much credit for wrong test selection but correct execution of the wrong test?
  • How to score conditions — all or nothing, or partial credit?
  • Conclusion without context — full deduction or partial?

Discuss these before grading starts.

Teaching Exercises

NoteSuggested Classroom Demonstrations

Test Selection Flowchart (Lesson 26 Review)

Build a decision flowchart on the board as a class:

How many samples?
├── One sample
│   ├── Mean? → One-sample t
│   └── Proportion? → One-proportion z
└── Two samples
    ├── Paired? → Paired t
    └── Independent?
        ├── Means? → Two-sample t (Welch)
        └── Proportions? → Two-proportion z

Then work through 6–8 scenarios where cadets must navigate the flowchart and set up (but not necessarily solve) each test. Focus on the identification, not the arithmetic.

“Find the Error” Activity (Lesson 26 Review)

Present 3–4 completed hypothesis tests, each with one subtle error:

  • Hypotheses written with \(\bar{x}\) instead of \(\mu\)
  • Missing conditions check
  • Conclusion says “accept \(H_0\)”
  • Used \(z\) when should have used \(t\)
  • Paired data analyzed as two-sample

Cadets work in pairs to identify each error. Excellent WPR preparation and directly assesses SLO 6.
