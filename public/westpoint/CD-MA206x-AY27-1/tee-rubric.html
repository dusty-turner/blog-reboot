Problem 1 — Data Fundamentals & Probability (30 points)

1a — Poisson Probability (5 pts)

Points Criteria
2 Identifies Poisson with \(\lambda = 5\)
3 \(P(X=3) = \frac{e^{-5}(5)^3}{3!} = \frac{125e^{-5}}{6} \approx 0.1404\)

1b — Poisson Complement (5 pts)

Points Criteria
2 Uses complement: \(P(X \geq 8) = 1 - P(X \leq 7)\)
3 Correct computation: \(\approx 1 - 0.8666 = 0.1334\) (accept reasonable values from table/formula)

1c — Normal Probability (5 pts)

Points Criteria
2 \(Z = (28 - 22)/4 = 1.5\); sketch with right tail shaded
3 \(P(Z > 1.5) = 1 - 0.9332 = 0.0668\)

1d — Normal Quantile (5 pts)

Points Criteria
2 \(z_{0.95} = 1.645\)
3 \(x = 22 + 1.645(4) = 28.58\) mph

1e — CLT Application (5 pts)

Points Criteria
2 \(\bar{X} \sim N(22, 4^2/40)\); \(SE = 4/\sqrt{40} = 0.632\)
2 \(Z = (23.1 - 22)/0.632 = 1.739\); \(P(\bar{X} > 23.1) = P(Z > 1.739) \approx 0.041\)
1 States assumption: random sample, \(n = 40 \geq 30\) so CLT applies

1f — Reasoning (5 pts)

Points Criteria
5 Yes — only a 4.1% chance of seeing \(\bar{x} \geq 23.1\) if the true mean were 22. This is unlikely, suggesting the true mean may exceed 22 mph. (Essentially an informal hypothesis test.)
3 Correct direction but weak reasoning
1 Conclusion without connection to probability

Problem 2 — Bayes’ Theorem (30 points)

2a — Tree Diagram (5 pts)

Points Criteria
5 Complete tree: branches for condition (0.02) and no condition (0.98), then test results with correct conditional probabilities (0.95/0.05 and 0.10/0.90)
3 Partial tree — missing one branch or wrong probability

2b — Bayes’ Theorem (8 pts)

Points Criteria
2 \(P(+) = P(+|C)P(C) + P(+|C^c)P(C^c) = 0.95(0.02) + 0.10(0.98) = 0.019 + 0.098 = 0.117\)
3 \(P(C|+) = \frac{P(+|C)P(C)}{P(+)} = \frac{0.019}{0.117}\)
3 \(= 0.1624\) (≈ 16.2%)

2c — Interpretation (5 pts)

Points Criteria
5 Only about 16% of positive tests are true positives. The test has an ~84% false positive rate. As a standalone diagnostic, this is poor — most positive results are false alarms.
3 Recognizes high false positive rate but incomplete explanation

2d — Sequential Testing (7 pts)

Points Criteria
3 After first positive: \(P(C|+_1) = 0.1624\); this becomes the new prior
4 \(P(C|+_1, +_2) = \frac{0.95 \times 0.1624}{0.95(0.1624) + 0.10(0.8376)} = \frac{0.1543}{0.1543 + 0.0838} = \frac{0.1543}{0.2381} \approx 0.648\)

Accept equivalent computation using full tree from scratch

2e — Base Rate Discussion (5 pts)

Points Criteria
5 Yes, much better (~65% vs ~16%). The low base rate (2%) means even a good test produces many false positives in the first round. The second test dramatically improves PPV because the “prior” after the first positive is much higher (16.2% vs 2%).
3 Correct conclusion but doesn’t explain the base rate mechanism

Points Criteria

Points Criteria

Points Criteria

3d — E(X) and SD(X) (5 pts)

Points Criteria

1 \(SD(X) = \sqrt{20(0.15)(0.85)} = \sqrt{2.55} \approx 1.597\)
3 Interpretation: “On average, the analyst expects 3 of the 20 channels to carry a signal of interest during a given time window.”

3e — Compound Probability (5 pts)

Points Criteria

3 \(E = 1.703 \times \frac{0.08}{\sqrt{28}} = 1.703 \times 0.01512 = 0.02574\)
4 CI: \(0.42 \pm 0.026 = (0.394, 0.446)\)

4c — Interpretation (7 pts)

Points Criteria

5 Mostly correct, minor language issue

4d — Informal HT via CI (5 pts)

Points Criteria
5 0.38 is below the CI lower bound (0.394), so it falls outside the 90% CI. This suggests the fleet consumes significantly more fuel than the expected 0.38 gal/mile at the 10% significance level.
3 Correct observation but no connection to inference

4e — Proportion CI (8 pts)

Points Criteria

Problem 5 — One-Sample t-Test (30 points)

5a — Hypotheses (5 pts)

Points Criteria
3 \(H_0: \mu = 3200\); \(H_a: \mu > 3200\) (one-tailed, right)
2 Correct justification for one-tailed (dietitian claims intake exceeds recommendation)

5b — Conditions (5 pts)

Points Criteria
2 Random sample; independence
3 \(n = 20 < 30\) — must assume population of caloric intake is approximately normal

5c — Test Statistic & p-value (8 pts)

Points Criteria
3 \(SE = 380/\sqrt{20} = 84.97\)
3 \(t = (3450 - 3200)/84.97 = 250/84.97 = 2.942\); \(df = 19\)
2 p-value: 1 - pt(2.942, 19) \(\approx 0.0042\)

5d — Decision (5 pts)

Points Criteria
2 Reject \(H_0\) (\(0.0042 < 0.01\))
3 “At \(\alpha = 0.01\), there is sufficient evidence that the mean daily caloric intake of cadets during CBT exceeds 3,200 calories.”

5e — CI and Connection (7 pts)

Points Criteria
4 \(3450 \pm 2.861(84.97) = 3450 \pm 243.1 = (3206.9, 3693.1)\)
3 3200 is below the CI lower bound → consistent with rejecting \(H_0\). Since 3200 is not in the 99% CI, we reject at \(\alpha = 0.01\).

Points Criteria

Problem 8 — Two-Proportion z-Test (30 points)

8a — Hypotheses (5 pts)

Points Criteria
5 \(H_0: p_A = p_B\); \(H_a: p_A \neq p_B\) (two-tailed)

8b — Conditions (5 pts)

Points Criteria
5 Random samples; independent brigades; success/failure counts \(\geq 10\) for each group (\(n_A\hat{p}_A = 28\), \(n_A(1-\hat{p}_A) = 322\), etc.)

8c — Computation (8 pts)

Points Criteria
2 \(\hat{p} = (28+42)/(350+400) = 70/750 = 0.0933\)
3 \(SE = \sqrt{0.0933(0.9067)(1/350 + 1/400)} = \sqrt{0.0846 \times 0.00536} = \sqrt{0.000453} = 0.02129\)
3 \(z = \frac{(0.080 - 0.105)}{0.02129} = \frac{-0.025}{0.02129} = -1.174\); p-value (two-tailed) \(\approx 0.240\)

8d — Decision (5 pts)

Points Criteria
2 Fail to reject \(H_0\)
3 “At \(\alpha = 0.05\), there is insufficient evidence of a difference in injury rates between Brigade A and Brigade B.”

8e — CI and Connection (7 pts)

Points Criteria
4 Unpooled CI: \((0.080 - 0.105) \pm 1.96 \sqrt{\frac{0.080(0.920)}{350} + \frac{0.105(0.895)}{400}}\); result includes 0
3 CI contains 0 → consistent with failing to reject; cannot rule out equal injury rates

Problem 10 — ANOVA (50 points)

10a — Hypotheses (5 pts)

Points Criteria
3 \(H_0: \mu_A = \mu_B = \mu_C\)
2 \(H_a\): At least one mean differs (NOT “all are different”)

10b — ANOVA Table (10 pts)

Points Criteria
2 \(df_B = k - 1 = 2\); \(df_W = N - k = 42\); \(df_T = 44\)
3 \(MSB = 267.6/2 = 133.8\)
3 \(MSW = 2268.0/42 = 54.0\)
2 \(F = 133.8/54.0 = 2.478\)

10c — Conditions (5 pts)

Points Criteria
2 Random assignment; independent groups
1 Normality within each group (approximate, or \(n = 15\) is moderate)
2 Equal variances: largest SD (\(s_C = 8.1\)) is less than twice smallest (\(s_B = 6.8\)) ✓

10d — p-value and Decision (5 pts)

Points Criteria
3 1 - pf(2.478, 2, 42) \(\approx 0.096\)
2 Fail to reject \(H_0\) at \(\alpha = 0.05\)

10e — Conclusion (5 pts)

Points Criteria
3 “At \(\alpha = 0.05\), there is insufficient evidence to conclude that the mean max push-ups differ among the three PT programs.”
2 Notes what ANOVA doesn’t tell us: even if significant, it wouldn’t tell us which programs differ (that requires post-hoc tests)

10f — Tukey Interpretation (8 pts)

Points Criteria
3 B vs A: significant (\(p = 0.048 < 0.05\)) — Program B produces significantly higher push-up scores than Program A
3 C vs A: not significant (\(p = 0.154\)); C vs B: not significant (\(p = 0.831\))
2 Summary: only B and A are significantly different; B and C are statistically indistinguishable, as are A and C

Note: Tukey results may seem inconsistent with the overall F-test (which was borderline). This is worth discussing — the overall F is a more conservative test, and the specific B-A comparison is driving the signal.

10g — Why Not Multiple t-Tests (5 pts)

Points Criteria
5 Multiple t-tests inflate the Type I error rate (family-wise error). With 3 groups, there are 3 pairwise tests. At \(\alpha = 0.05\) each, the overall error rate is \(1 - (0.95)^3 = 14.3\%\), not 5%. ANOVA tests all groups simultaneously at a single \(\alpha\).
3 Mentions inflated error rate but doesn’t quantify or explain

10h — Recommendation (7 pts)

Points Criteria
3 References the statistical evidence: overall ANOVA was not significant at \(\alpha = 0.05\), but Tukey HSD showed B significantly outperformed A
2 Recommendation with justification (e.g., “Program B shows promise over traditional training, but the evidence is borderline; consider a larger study”)
2 Acknowledges limitations: sample size, duration of study, generalizability beyond this company
