Problem 1 — CLT & Confidence Intervals (40 points)

1a — Conditions (8 pts)

Points Criteria
3 Random sample: States the sample is random (given in problem)
2 Independence: Reasonable to assume 36 cadets are less than 10% of population
3 Sample size: \(n = 36 \geq 30\), so CLT applies; OR population approximately normal

Deduct 1 pt per condition not explicitly stated; deduct 2 pts per condition stated but not verified

1b — CI Construction (10 pts)

Points Criteria
3 Identifies t-procedure with \(df = 35\); finds \(t^* = 2.030\) (or close value from table)
3 Margin of error: \(E = 2.030 \times \frac{2.4}{\sqrt{36}} = 2.030 \times 0.4 = 0.812\)
4 CI: \(7.8 \pm 0.812 = (6.988, 8.612)\); or equivalent with minor rounding differences

Partial credit: 5 pts for correct formula and setup with arithmetic error; 2 pts for using z instead of t (wrong procedure but correct execution)

1c — Interpretation (7 pts)

Points Criteria
7 “We are 95% confident that the true mean PT hours per week for all cadets is between 6.99 and 8.61 hours” — must include confidence level, “true mean” (parameter), context, and both bounds
5 Mostly correct but missing one element (e.g., no context or says “average” instead of “true mean”)
3 Incorrect interpretation: “95% chance” or “95% of cadets”
0 Missing or fundamentally wrong

1d — Sample Size (5 pts)

Points Criteria
5 \(n = \left(\frac{z_{\alpha/2} \cdot s}{E}\right)^2 = \left(\frac{1.96 \times 2.4}{0.5}\right)^2 = (9.408)^2 = 88.5 \Rightarrow n = 89\) (rounds up)
3 Correct formula but arithmetic error or forgot to round up
1 Uses \(t^*\) instead of \(z^*\) (common approach, give partial)

1e — CI Width Explanation (5 pts)

Points Criteria
5 Higher confidence → larger critical value (\(z^*\) or \(t^*\)) → larger margin of error → wider interval. Connects to needing to “cast a wider net” to be more confident.
3 Correct direction (wider) with partial explanation
1 Just states “wider” without explanation

1f — z vs t Justification (5 pts)

Points Criteria
3 t-procedure because \(\sigma\) is unknown (using \(s = 3.1\)) and \(n = 16 < 30\)
2 States additional assumption: population of PT hours is approximately normally distributed (needed because \(n < 30\))

Deduct 2 pts for choosing z-procedure; 1 pt for no mention of normality assumption

Problem 2 — One-Sample t-Test (35 points)

2a — Hypotheses (5 pts)

Points Criteria
3 \(H_0: \mu = 15.5\) and \(H_a: \mu < 15.5\) (one-tailed, left)
2 States one-tailed with justification (instructor believes cadets are faster, meaning lower times)

Deduct 2 pts for wrong direction; 1 pt for using \(\bar{x}\) instead of \(\mu\)

2b — Conditions (5 pts)

Points Criteria
2 Random sample (stated in problem)
1 Independence (25 cadets < 10% of company/population)
2 \(n = 25\) — need to assume population is approximately normal (or accept \(n\) close to 30)

2c — Test Statistic (10 pts)

Points Criteria
3 Identifies one-sample t-test with \(df = 24\)
3 \(SE = s/\sqrt{n} = 1.6/\sqrt{25} = 1.6/5 = 0.32\)
4 \(t = \frac{14.8 - 15.5}{0.32} = \frac{-0.7}{0.32} = -2.1875\)

Partial credit: 5 pts for correct formula with arithmetic error

2d — p-value (5 pts)

Points Criteria
3 States using pt(-2.1875, df=24) or equivalent
2 p-value ≈ 0.0194 (one-tailed)

Accept values in range 0.018–0.021 depending on rounding of test stat

2e — Decision and Conclusion (5 pts)

Points Criteria
2 Decision: Reject \(H_0\) (since \(0.0194 < 0.05\))
3 Conclusion: “At \(\alpha = 0.05\), there is sufficient evidence to conclude that the mean 2-mile run time for cadets in this company is less than 15.5 minutes.”

Deduct 2 pts for no context; 1 pt for saying “accept \(H_a\)” instead of “reject \(H_0\)”

2f — CI-HT Duality (5 pts)

Points Criteria
5 The CI is (14.14, 15.46) and does not contain \(\mu_0 = 15.5\). This is consistent with rejecting \(H_0\) at \(\alpha = 0.05\). When the hypothesized value falls outside the 95% CI, we reject at the 5% level.
3 Correct observation but incomplete explanation of the connection
1 States “consistent” or “not consistent” without justification

Problem 3 — Two-Sample t-Test (30 points)

3a — Hypotheses (5 pts)

Points Criteria
3 \(H_0: \mu_A = \mu_B\) (or \(\mu_A - \mu_B = 0\)); \(H_a: \mu_A \neq \mu_B\) (two-tailed)
2 Justification for two-tailed: researcher wants to detect a difference in either direction; no prior theory about which method is better

Accept \(H_a: \mu_A > \mu_B\) (one-tailed) if well-justified; deduct 1 pt for no justification

3b — Conditions (5 pts)

Points Criteria
2 Random assignment to groups; groups are independent
1 Sample sizes (\(n_A = 18\), \(n_B = 22\)): need normality assumption or near-30 argument
2 Welch (not pooled) because we cannot assume \(\sigma_A = \sigma_B\) (\(s_A = 6.3 \neq s_B = 7.8\))

3c — Test Statistic (10 pts)

Points Criteria
4 \(SE = \sqrt{\frac{6.3^2}{18} + \frac{7.8^2}{22}} = \sqrt{\frac{39.69}{18} + \frac{60.84}{22}} = \sqrt{2.205 + 2.765} = \sqrt{4.970} = 2.230\)
4 \(t = \frac{82.4 - 78.1}{2.230} = \frac{4.3}{2.230} = 1.928\)
2 States degrees of freedom (Welch approximation, \(df \approx 37.6\), accept any reasonable value)

Partial credit: 5 pts for correct formula with arithmetic error

3d — Decision and Conclusion (5 pts)

Points Criteria
2 Decision: Fail to reject \(H_0\) (since \(0.0632 > 0.05\))
3 Conclusion: “At \(\alpha = 0.05\), there is insufficient evidence to conclude that there is a significant difference in mean exam scores between Method A and Method B.”

Deduct 2 pts for no context; deduct all if says “accept \(H_0\)”

3e — CI Interpretation (5 pts)

Points Criteria
2 The CI \((-0.25, 8.85)\) contains 0
3 Since 0 is in the interval, we cannot rule out no difference between the means. This is consistent with failing to reject \(H_0\).

Problem 4 — Paired t-Test (30 points)

4a — Paired Justification (5 pts)

Points Criteria
5 Same soldiers measured twice (before and after) — measurements are dependent/linked. A two-sample test would treat them as independent, ignoring the pairing.
3 Says “same subjects” but doesn’t explain why this matters
1 Identifies as paired without explanation

4b — Differences (5 pts)

Points Criteria
2 Differences \(d_i\): 1.8, 1.3, 3.2, 0.2, 2.3, 0.6, 2.5, 1.7, 1.8, 2.6
2 \(\bar{d} = 1.80\)
1 \(s_d \approx 0.896\)

Partial credit: 1 pt for correct direction of subtraction but arithmetic errors

4c — Hypotheses (5 pts)

Points Criteria
5 \(H_0: \mu_d = 0\); \(H_a: \mu_d > 0\) (where \(d = \text{Before} - \text{After}\), so \(\mu_d > 0\) means reduction)

Deduct 2 pts for wrong direction; accept \(H_a: \mu_d < 0\) if \(d = \text{After} - \text{Before}\) is defined

4d — Test Statistic (5 pts)

Points Criteria
2 \(SE = s_d/\sqrt{n} = 0.896/\sqrt{10} = 0.283\)
3 \(t = \frac{1.80 - 0}{0.283} = 6.357\) with \(df = 9\)

4e — p-value and Conclusion (5 pts)

Points Criteria
2 p-value from 1 - pt(6.357, df=9) ≈ 0.00007 (essentially 0)
3 “At \(\alpha = 0.05\), there is strong evidence that the stress management program reduces cortisol levels in soldiers.”

4f — Confidence Interval (5 pts)

Points Criteria
3 \(\bar{d} \pm t^* \cdot \frac{s_d}{\sqrt{n}} = 1.80 \pm 2.262(0.283) = 1.80 \pm 0.641 = (1.16, 2.44)\)
2 “We are 95% confident the true mean reduction in cortisol is between 1.16 and 2.44 ng/mL.”

Problem 5A — One-Proportion z-Test (20 points)

5Aa — Hypotheses (5 pts)

Points Criteria
5 \(H_0: p = 0.85\); \(H_a: p < 0.85\)

Deduct 2 pts for using \(\hat{p}\) instead of \(p\)

5Ab — Conditions (5 pts)

Points Criteria
2 Random sample (stated)
3 \(np_0 = 120(0.85) = 102 \geq 10\); \(n(1-p_0) = 120(0.15) = 18 \geq 10\)

5Ac — Test Statistic (5 pts)

Points Criteria
2 \(\hat{p} = 96/120 = 0.80\)
3 \(z = \frac{0.80 - 0.85}{\sqrt{0.85(0.15)/120}} = \frac{-0.05}{0.0326} = -1.534\)

Important: SE uses \(p_0 = 0.85\), not \(\hat{p}\). Deduct 2 pts for using \(\hat{p}\) in SE.

5Ad — Decision and Conclusion (5 pts)

Points Criteria
2 Fail to reject \(H_0\) (p-value \(= 0.0808 > 0.05\))
3 “At \(\alpha = 0.05\), there is insufficient evidence to conclude that the IOCT pass rate has dropped below 85%.”

Problem 5B — Two-Proportion z-Test (20 points)

5Be — Hypotheses (5 pts)

Points Criteria
5 \(H_0: p_1 = p_2\); \(H_a: p_1 \neq p_2\) (two-tailed)

5Bf — Pooled Proportion and Test Stat (5 pts)

Points Criteria
2 \(\hat{p} = \frac{45 + 62}{150 + 180} = \frac{107}{330} = 0.3242\)
3 \(z = \frac{(0.300 - 0.344)}{\sqrt{0.3242(0.6758)(1/150 + 1/180)}} = \frac{-0.044}{0.0517} = -0.993\) (accept reasonable rounding)

5Bg — Decision and Conclusion (5 pts)

Points Criteria
2 Fail to reject \(H_0\) (\(0.3204 > 0.05\))
3 “At \(\alpha = 0.05\), there is insufficient evidence to conclude a difference in the proportion of cadets using AEC tutoring between the two regiments.”

5Bh — CI and Connection (5 pts)

Points Criteria
3 CI using unpooled SE: \((0.300 - 0.344) \pm 1.96\sqrt{\frac{0.300(0.700)}{150} + \frac{0.344(0.656)}{180}} \approx (-0.044) \pm 0.102 = (-0.146, 0.058)\)
2 CI contains 0, consistent with failing to reject — we cannot rule out that the proportions are equal
