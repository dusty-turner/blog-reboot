Problem 1 — Data Fundamentals (25 points)

1a — Variable Identification (5 pts)

Points Criteria
5 Correctly identifies explanatory (sleep group — categorical) and response (reaction time — quantitative) with correct classification of each
3 Correct identification of both variables but missing or incorrect classification
2 Only one variable correctly identified and classified
0 Both incorrect or missing

1b — Study Design (5 pts)

Points Criteria
5 Correctly identifies as experiment with justification (random assignment to treatment groups)
3 Correct identification but weak or missing justification
1 Incorrect identification (says observational)
0 Missing

1c — Causal Conclusion (5 pts)

Points Criteria
5 Yes, causal conclusion is possible; justified by random assignment (experiment) and control of confounding variables
3 Correct answer (yes) but incomplete justification
1 Incorrect answer (no) but shows some understanding of causation
0 Missing or no understanding demonstrated

1d — Five-Number Summary Analysis (10 pts)

Points Criteria
3 IQR correctly computed (Q3 − Q1 = 280 − 235 = 45)
4 Outlier test correct: upper fence = Q3 + 1.5(IQR) = 280 + 67.5 = 347.5; 340 < 347.5 → not an outlier; work shown
3 Mean vs median: correctly identifies median as more appropriate with justification (e.g., potential right skew based on five-number summary, or distance from Q3 to Max vs Min to Q1)

Partial credit: 1 pt for each correct component without complete work shown

Problem 2 — Probability (30 points)

2a — Law of Total Probability (5 pts)

Points Criteria
5 Correct setup and answer: \(P(G) = 0.60(0.05) + 0.30(0.10) + 0.10(0.20) = 0.030 + 0.030 + 0.020 = 0.080\)
3 Correct formula setup but arithmetic error
1 Incorrect formula (e.g., adds probabilities without weighting)
0 Missing or fundamentally wrong

2b — Bayes’ Theorem (10 pts)

Points Criteria
10 Correct application: \(P(I|G) = \frac{P(G|I)P(I)}{P(G)} = \frac{0.20 \times 0.10}{0.08} = 0.25\); formula stated, values substituted, correct answer
7 Correct formula and setup but arithmetic error or used wrong \(P(G)\) from part (a)
4 Correct formula stated but wrong values substituted
2 Attempted but used wrong approach (e.g., computed \(P(G|I)\) instead)
0 Missing

2c — Independence (5 pts)

Points Criteria
5 Checks \(P(G|I) = 0.20 \neq P(G) = 0.08\), therefore not independent; or checks \(P(G \cap I) = 0.02 \neq P(G)P(I) = 0.008\)
3 Correct conclusion with partial justification
1 Correct conclusion but no mathematical justification
0 Incorrect or missing

2d — Binomial Application (10 pts)

Points Criteria
3 Identifies Binomial distribution with \(n = 8\), \(p = 0.05\)
2 States/checks BINS conditions
5 Correctly computes \(P(X \leq 1) = P(X=0) + P(X=1)\); \(P(X=0) = \binom{8}{0}(0.05)^0(0.95)^8\) and \(P(X=1) = \binom{8}{1}(0.05)^1(0.95)^7\); correct answer ≈ 0.9428

Partial credit: 2 pts for correct setup of either \(P(X=0)\) or \(P(X=1)\) even with arithmetic error

Problem 3 — Discrete Random Variables (30 points)

3a — Distribution Identification (5 pts)

Points Criteria
5 Identifies Binomial\((n=4, p=0.70)\); verifies BINS: Binary (hit/miss), Independent rounds, \(n=4\) fixed, \(p=0.70\) constant
3 Correct distribution and parameters but incomplete conditions
1 Correct distribution without parameters or conditions

3b — Probability Computation (10 pts)

Points Criteria
5 \(P(X=3) = \binom{4}{3}(0.70)^3(0.30)^1 = 4(0.343)(0.30) = 0.4116\)
5 \(P(X \geq 3) = P(X=3) + P(X=4) = 0.4116 + (0.70)^4 = 0.4116 + 0.2401 = 0.6517\)

Partial credit: 3 pts for correct formula setup with arithmetic error; 1 pt for identifying correct approach (complement or direct)

3c — Expected Value and SD (10 pts)

Points Criteria
3 \(E(X) = np = 4(0.70) = 2.8\)
3 \(\text{SD}(X) = \sqrt{np(1-p)} = \sqrt{4(0.70)(0.30)} = \sqrt{0.84} \approx 0.917\)
4 Interpretation: “On average, a cadet hits the target 2.8 out of 4 times” or equivalent contextual statement

Deductions: −2 for missing context in interpretation; −1 for missing units or rounding errors

3d — Total Expected Hits (5 pts)

Points Criteria
5 \(E(\text{total}) = 12 \times E(X) = 12 \times 2.8 = 33.6\) hits; uses linearity of expectation
3 Correct answer but no justification
0 Missing or incorrect approach

Problem 4A — Continuous RV (20 points)

4Aa — Find \(c\) (5 pts)

Points Criteria
5 \(\int_0^3 cx^2\,dx = 1 \Rightarrow c \cdot \frac{x^3}{3}\Big|_0^3 = 1 \Rightarrow c \cdot 9 = 1 \Rightarrow c = 1/9\)
3 Correct integral setup but error in evaluation
1 Knows to set integral equal to 1 but can’t evaluate

4Ab — CDF (5 pts)

Points Criteria
5 \(F(x) = \int_0^x \frac{t^2}{9}\,dt = \frac{x^3}{27}\) for \(0 \leq x \leq 3\); with \(F(x) = 0\) for \(x < 0\) and \(F(x) = 1\) for \(x > 3\)
3 Correct integral but missing piecewise definition
1 Setup only

4Ac — Probability (5 pts)

Points Criteria
5 \(P(1 \leq X \leq 2) = F(2) - F(1) = \frac{8}{27} - \frac{1}{27} = \frac{7}{27} \approx 0.259\)
3 Correct method but arithmetic error
1 Correct setup (integral from 1 to 2) but can’t evaluate

4Ad — Expected Value (5 pts)

Points Criteria
5 \(E(X) = \int_0^3 x \cdot \frac{x^2}{9}\,dx = \frac{1}{9} \cdot \frac{x^4}{4}\Big|_0^3 = \frac{81}{36} = 2.25\)
3 Correct formula \(E(X) = \int x f(x)\,dx\) but error in computation
1 Correct formula stated

Problem 4C — Exponential Distribution (10 points)

4Ch — Time Between Arrivals (5 pts)

Points Criteria
2 Identifies Exponential with \(\lambda = 6\) per hour (or equivalently, mean \(= 1/6\) hour \(= 10\) min)
3 \(P(X > 0.25) = e^{-6(0.25)} = e^{-1.5} \approx 0.223\) (converts 15 min to 0.25 hr)

Common error: forgetting to convert minutes to hours — deduct 2 pts

4Ci — Memoryless Property (5 pts)

Points Criteria
2 Names the memoryless property
3 \(P(X > 15 | X > 10) = P(X > 5) = e^{-6(5/60)} = e^{-0.5} \approx 0.607\); or correctly states that elapsed time is irrelevant

Problem 6 — Poisson (25 points)

6a — Exact Probability (5 pts)

Points Criteria
5 \(P(X=2) = \frac{e^{-4}(4)^2}{2!} = \frac{e^{-4} \cdot 16}{2} = 8e^{-4} \approx 0.1465\); formula and computation shown
3 Correct formula but arithmetic error

6b — P(X = 0) (5 pts)

Points Criteria
5 \(P(X=0) = e^{-4} \approx 0.0183\)
3 Formula correct, arithmetic error

6c — Complement (5 pts)

Points Criteria
5 \(P(X \geq 3) = 1 - P(X \leq 2) = 1 - [P(X=0) + P(X=1) + P(X=2)]\); correct computation ≈ \(1 - 0.2381 = 0.7619\)
3 Correct approach (complement) but arithmetic error
1 Tries to enumerate \(P(X=3) + P(X=4) + \cdots\) without using complement

6d — Rate Adjustment (5 pts)

Points Criteria
2 Correctly adjusts: \(\lambda = 4 \times 2 = 8\) for two weeks
3 \(P(X=10) = \frac{e^{-8}(8)^{10}}{10!} \approx 0.0993\)

Deduct 3 pts for using \(\lambda = 4\) without adjustment

6e — Mean and SD Interpretation (5 pts)

Points Criteria
1 Mean = 4 incidents per week
1 SD = \(\sqrt{4} = 2\) incidents per week
3 Interpretation in context: “The number of weekly incidents typically varies by about 2 incidents from the average of 4” or equivalent

Deduct 2 pts for no context in interpretation
