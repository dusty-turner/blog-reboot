Lesson Deep Dive

Lesson 28 — Simple Linear Regression I

Objectives (SLOs 1, 4):

  • Fit a simple linear regression model
  • Interpret slope and intercept in context
  • Assess model fit using \(R^2\)
TipKey Concepts
  • Model: \(\hat{y} = b_0 + b_1 x\) where \(b_1 = r \cdot \frac{s_y}{s_x}\) and \(b_0 = \bar{y} - b_1 \bar{x}\)
  • Slope interpretation: “For each one-unit increase in \(x\), \(\hat{y}\) changes by \(b_1\) units, on average”
  • Intercept interpretation: “When \(x = 0\), we predict \(\hat{y} = b_0\)” — but only if \(x = 0\) is within the range of the data
  • \(R^2\): Proportion of variability in \(y\) explained by the linear relationship with \(x\)
  • Correlation (\(r\)): Measures strength and direction of linear relationship
ImportantGrading Focus
  • Slope interpretation: Must include “for each one-unit increase in [x in context]” and “on average.” Missing either is incomplete.
  • Intercept context: If \(x = 0\) is outside the data range, the intercept has no practical interpretation — cadets should state this.
  • \(R^2\) interpretation: “\(R^2 = 0.72\) means 72% of the variability in [y in context] is explained by the linear relationship with [x in context].” Not “72% of the data is explained.”
  • Extrapolation: Predictions outside the range of \(x\) data are extrapolation and unreliable. Cadets must flag this.
NoteDeeper Material

Least squares derivation: The OLS estimates minimize \(\sum (y_i - \hat{y}_i)^2\). Taking partial derivatives with respect to \(b_0\) and \(b_1\) and setting to zero gives the normal equations. This connects to calculus (optimization) and linear algebra (\(\hat{\beta} = (X^TX)^{-1}X^Ty\)). Faculty should understand both perspectives.

Correlation is not causation: This is the most important phrase in statistics. Even a strong correlation (\(r = 0.95\)) between two variables does not establish causation. Only randomized experiments can do that (connecting back to Lesson 2). Examples: ice cream sales and drowning deaths (confounder: temperature); shoe size and reading ability (confounder: age).

Anscombe’s quartet: Four datasets with identical regression statistics (\(\bar{x}\), \(\bar{y}\), \(r\), \(b_0\), \(b_1\), \(R^2\)) but completely different scatterplots. This demonstrates why you must always plot the data before fitting a model. The Datasaurus Dozen is a modern extension.

Lesson 30 — Simple Linear Regression II

Objectives (SLOs 1, 4, 5, 6):

  • Conduct inference on regression coefficients
  • Check regression conditions (LINE)
  • Make predictions and interpret prediction intervals
TipKey Concepts
  • LINE conditions: Linearity, Independence, Normality of residuals, Equal variance (constant spread in residual plot)
  • Inference on slope: \(t = \frac{b_1 - 0}{SE_{b_1}}\) tests \(H_0: \beta_1 = 0\) (is there a linear relationship?)
  • CI for slope: \(b_1 \pm t^* \cdot SE_{b_1}\)
  • Prediction interval vs confidence interval: Prediction interval for a new observation is wider than CI for the mean response
ImportantGrading Focus
  • LINE conditions: Every regression analysis must include a conditions check. Residual plot (residuals vs fitted) checks L and E; normal probability plot checks N; study design addresses I.
  • Residual plot interpretation: “No pattern = good.” Curvature → linearity violated; fan shape → equal variance violated. Cadets should sketch or reference the actual residual plot.
  • \(H_0: \beta_1 = 0\) interpretation: “There is no linear relationship between \(x\) and \(y\)” — rejecting means the slope is significantly different from zero.
  • Prediction vs confidence band: Prediction intervals are wider because they account for individual variation plus estimation uncertainty.
NoteDeeper Material

Residual analysis in depth: Residuals \(e_i = y_i - \hat{y}_i\) should be approximately iid \(N(0, \sigma^2)\) if the model is correct. Patterns in residual plots diagnose specific problems:

  • Curvature: Try a polynomial term or transformation
  • Fan shape (heteroscedasticity): Try a log transformation or weighted least squares
  • Outliers: High leverage (extreme \(x\)) vs high influence (changes the fit). Cook’s distance measures influence.

\(R^2\) limitations: \(R^2\) always increases when you add predictors (even useless ones). This is why adjusted \(R^2\) exists: \(R^2_{\text{adj}} = 1 - \frac{(1-R^2)(n-1)}{n-p-1}\) where \(p\) is the number of predictors. Adjusted \(R^2\) penalizes for model complexity.

Gauss-Markov Theorem: Under the LINE assumptions, OLS estimators are BLUE (Best Linear Unbiased Estimators). “Best” means minimum variance. This is why OLS is the default — it’s optimal under standard conditions.

Lesson 31 — Multiple Linear Regression I

Objectives (SLOs 1, 4, 6):

  • Extend SLR to multiple predictors
  • Interpret coefficients in MLR context
  • Compare models using adjusted \(R^2\)
TipKey Concepts
  • Model: \(\hat{y} = b_0 + b_1 x_1 + b_2 x_2 + \cdots + b_p x_p\)
  • Coefficient interpretation: “\(b_1\) is the expected change in \(y\) for a one-unit increase in \(x_1\), holding all other predictors constant”
  • Adjusted \(R^2\): Accounts for the number of predictors; use for model comparison
  • Categorical predictors: Represented by indicator (dummy) variables
ImportantGrading Focus
  • “Holding all other variables constant”: This phrase is mandatory in every MLR coefficient interpretation. Without it, the interpretation is fundamentally wrong.
  • Adjusted \(R^2\) vs \(R^2\): For model comparison with different numbers of predictors, use adjusted \(R^2\). Cadets who compare raw \(R^2\) values across models with different predictors are making an error.
  • Indicator variables: For a categorical variable with \(k\) levels, we need \(k-1\) indicators. The omitted level is the reference category. The coefficient is the difference from the reference group.
NoteDeeper Material

Multicollinearity: When predictors are highly correlated, coefficient estimates become unstable (large SEs), sign flips can occur, and individual predictors may appear non-significant even though the overall model is significant. VIF (Variance Inflation Factor) > 10 is a common flag. This is a major practical issue in MLR that cadets will encounter in their projects.

Simpson’s Paradox in regression: Adding a predictor can reverse the sign of another predictor’s coefficient. Example: hospital size appears positively associated with mortality (bigger hospitals → more deaths). But after controlling for case severity, the relationship reverses (bigger hospitals handle harder cases). This connects back to confounding in Lesson 2.

Partial F-tests: Testing whether a subset of predictors collectively improves the model. The full-vs-reduced model comparison uses \(F = \frac{(SSE_R - SSE_F)/(df_R - df_F)}{SSE_F/df_F}\). R does this with anova(reduced_model, full_model).

Teaching Exercises

NoteSuggested Classroom Demonstrations

Anscombe’s Quartet (Lesson 28)

Show all four Anscombe datasets. They all have identical summary statistics and regression lines, but wildly different scatterplots. Moral: always plot your data first.

Residual Plot Reading (Lesson 30)

Show 4–5 residual plots and have cadets diagnose each:

  • Random scatter → model OK
  • Curved pattern → linearity violated
  • Fan shape → non-constant variance
  • One extreme point → potential influential observation

This is a critical skill for both the WPR and the project.

“What does this coefficient mean?” Gallery Walk (Lesson 31)

Post 4–5 MLR output printouts around the room. Each has 2–3 predictors. Cadets circulate and write interpretations of each coefficient on sticky notes. Review as a class — correct any missing “holding all other variables constant” phrases.
