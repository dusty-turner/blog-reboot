Lesson Deep Dive

Lesson 9 — Discrete Random Variables

Objectives (SLOs 3, 7):

  • Understand how PMFs are created and used
  • Understand how CDFs are used
  • Calculate Expected Value and Variance
TipKey Concepts
  • Random Variable: A function mapping outcomes to numbers. Discrete RVs have countable support.
  • PMF: \(p(x) = P(X = x)\), must satisfy \(p(x) \geq 0\) and \(\sum p(x) = 1\)
  • CDF: \(F(x) = P(X \leq x)\), non-decreasing, right-continuous
  • Expected Value: \(E(X) = \sum x \cdot p(x)\) — the “long-run average”
  • Variance: \(\text{Var}(X) = E(X^2) - [E(X)]^2\)
ImportantGrading Focus
  • PMF validity: Cadets must verify both conditions (\(p(x) \geq 0\) and \(\sum p(x) = 1\)) when constructing a PMF
  • E(X) interpretation: “If we repeated this experiment many times, the average outcome would approach \(E(X)\)” — not “the next outcome will be \(E(X)\)”
  • Variance shortcut: \(\text{Var}(X) = E(X^2) - [E(X)]^2\) is faster than \(\sum (x - \mu)^2 p(x)\). Cadets must compute \(E(X^2)\) correctly — the most common error is squaring \(E(X)\) instead of computing \(\sum x^2 p(x)\).
NoteDeeper Material

Linearity of Expectation: \(E(aX + b) = aE(X) + b\). This holds even for dependent random variables (for the sum case: \(E(X + Y) = E(X) + E(Y)\)). This is arguably the most useful property in probability and statistics — it’s why the sample mean is unbiased.

Variance properties: \(\text{Var}(aX + b) = a^2 \text{Var}(X)\) — the additive constant drops out, and the multiplicative constant gets squared. This previews standardization: \(Z = (X - \mu)/\sigma\) has \(E(Z) = 0\) and \(\text{Var}(Z) = 1\).

MGFs (optional): The moment-generating function \(M_X(t) = E(e^{tX})\) uniquely determines a distribution. \(M'(0) = E(X)\) and \(M''(0) = E(X^2)\). Useful for proving the CLT but beyond course scope.

Lesson 10 — Binomial Distribution

Objectives (SLOs 3, 7):

  • Verify binomial conditions for counts of successes
  • Compute binomial PMF/CDF values
  • Interpret binomial mean and variance
TipKey Concepts
  • Binomial conditions (BINS): Binary outcomes, Independent trials, Number of trials fixed, Same probability of success
  • PMF: \(P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}\)
  • Mean: \(\mu = np\); Variance: \(\sigma^2 = np(1-p)\)
ImportantGrading Focus
  • BINS verification: Every binomial problem must start by checking all four conditions. Deduct if cadets skip this.
  • “At least” / “at most”: \(P(X \geq k) = 1 - P(X \leq k-1)\). Watch for off-by-one errors.
  • Context: “The probability of getting exactly 3 defective items out of 10” — cadets must define what “success” means in their setup.
NoteDeeper Material

Binomial → Poisson: When \(n\) is large, \(p\) is small, and \(np = \lambda\) is moderate, the Binomial\((n, p)\) is well-approximated by Poisson\((\lambda)\). This is the bridge to Lesson 11.

Binomial → Normal: When \(np \geq 10\) and \(n(1-p) \geq 10\), the Binomial is approximately Normal\((\mu = np, \sigma^2 = np(1-p))\). This previews the CLT and the conditions for proportion tests in Block II.

Hypergeometric vs Binomial: If sampling without replacement from a finite population, the correct model is Hypergeometric, not Binomial. The Binomial approximation is good when the sample is less than 10% of the population (the “10% condition” that reappears in inference).

Lesson 11 — Poisson Distribution

Objectives (SLO 7):

  • Model counts and rare events with Poisson distributions
  • Calculate Poisson probabilities and parameters
TipKey Concepts
  • Poisson setting: Counting events in a fixed interval (time, area, volume) where events occur independently at a constant average rate
  • PMF: \(P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}\) for \(k = 0, 1, 2, \ldots\)
  • Mean = Variance: \(E(X) = \text{Var}(X) = \lambda\)
ImportantGrading Focus
  • Poisson conditions: Events occur independently, at a constant rate, one at a time. Cadets should state these.
  • Rate adjustment: If \(\lambda = 3\) per hour, then over 2 hours \(\lambda = 6\). Cadets forget to scale the rate to the interval.
  • \(E(X) = \text{Var}(X) = \lambda\): This is a quick check — if the sample mean and variance are very different, the Poisson model may be inappropriate.
NoteDeeper Material

Poisson process: The Poisson distribution arises from a Poisson process where inter-arrival times are Exponential\((\lambda)\). This connects directly to Lesson 14 (Exponential). The number of events in time \(t\) is Poisson\((\lambda t)\), and the time between events is Exp\((\lambda)\).

Overdispersion: In practice, count data often has variance > mean (overdispersion). This violates the Poisson assumption and suggests a Negative Binomial model instead. Military examples: IED incidents per patrol (clustering violates independence), sick call visits per company (some companies have systematically higher rates).

Lesson 12 — Continuous Random Variables

Objectives (SLOs 3, 7):

  • Interpret PDFs and areas as probabilities
  • Compute probabilities from PDFs and CDFs
  • Find \(E(X)\) and \(\text{Var}(X)\) for continuous RVs
TipKey Concepts
  • PDF: \(f(x) \geq 0\) and \(\int_{-\infty}^{\infty} f(x)\,dx = 1\). Note: \(f(x) \neq P(X = x)\)!
  • Probability = Area: \(P(a \leq X \leq b) = \int_a^b f(x)\,dx\)
  • CDF: \(F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)\,dt\)
  • \(P(X = c) = 0\) for continuous RVs — this surprises cadets
  • \(E(X) = \int x f(x)\,dx\); \(\text{Var}(X) = \int (x - \mu)^2 f(x)\,dx = E(X^2) - [E(X)]^2\)
ImportantGrading Focus
  • PDF ≠ probability: \(f(x)\) can exceed 1; only areas are probabilities. This is the most common misconception.
  • Integration skills: Cadets must compute simple integrals (polynomials, basic exponentials). If integration is rusty, this lesson will be painful — recommend practice.
  • CDF from PDF: \(F(x) = \int_{-\infty}^{x} f(t)\,dt\) — cadets must adjust limits of integration based on the support of the distribution.
NoteDeeper Material

Quantile function: The inverse CDF, \(F^{-1}(p)\), gives the value \(x\) such that \(P(X \leq x) = p\). This is how we find critical values (\(z_{\alpha/2}\), \(t_{\alpha/2}\)) in Block II. Connecting the CDF to quantiles now makes the transition to inference smoother.

Median of continuous RV: The median \(m\) satisfies \(F(m) = 0.5\), equivalently \(\int_{-\infty}^{m} f(x)\,dx = 0.5\). For symmetric distributions, median = mean. For skewed distributions, they differ — connecting back to Lesson 3.

Mixed distributions: Some real-world phenomena have both discrete and continuous components (e.g., insurance claims: \(P(X = 0)\) is discrete, but \(X | X > 0\) is continuous). This is beyond scope but helps faculty understand why the discrete/continuous distinction isn’t always clean.

WPR I — SLO Discussion

CautionWPR I Coverage — Random Variables Block

Lessons 9–12 are core SLO 7 material. WPR I questions from this block should test:

  • Constructing or verifying a PMF, computing \(E(X)\) and \(\text{Var}(X)\)
  • Setting up and computing Binomial probabilities (with BINS check)
  • Setting up and computing Poisson probabilities (with rate adjustment)
  • Computing probabilities from a given PDF (integration)
  • Finding the CDF from a PDF and using it

Objective alignment check: All objectives map to SLO 7, with SLO 3 (variability) appearing through \(E(X)\) and \(\text{Var}(X)\) interpretation. Consider adding a question that requires interpreting expected value in context to strengthen SLO 1 coverage on WPR I.

Teaching Exercises

NoteSuggested Classroom Demonstrations

Binomial Simulation (Lesson 10)

Flip 10 coins as a class (or simulate in R). Record the number of heads. Repeat 20+ times. Plot the results and overlay the Binomial(10, 0.5) PMF. Cadets see the theoretical distribution emerge from data.

Poisson in the Wild (Lesson 11)

Have cadets count events in a real-time interval: cars passing a window in 2 minutes, typos on a page, etc. Compare the sample mean and variance — if they’re close, the Poisson model fits. If not, discuss why.

PDF Area Demo (Lesson 12)

Draw a triangle PDF on the board (e.g., \(f(x) = 2x\) for \(0 \leq x \leq 1\)). Have cadets compute \(P(0 \leq X \leq 0.5)\) by hand. Then show that \(f(0.5) = 1\) but \(P(X = 0.5) = 0\). This drives home “PDF ≠ probability.”
