More Probability

Compliments

For any event \(A\):

\[ P(A^c) = 1 - P(A) \]

Addition Rule

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B). \]

Mutually Exclusive

Two events are mutually exclusive when it is impossible for both to happen at the same time.

Therefore:

\[ P(A \cup B) = P(A) + P(B) \]

Independence

Two events \(A\) and \(B\) are independent if knowing that one occurs does not change the probability of the other.

\[ P(A \cap B) = P(A)P(B) \]

Equivalently (if \(P(B) > 0\)):

\[ P(A \mid B) = P(A) \]

Intuition: The outcome of one event gives no information about the other.

Does Mutually Exclusive imply independence or no or undetermined?

Conditional Probability

The probability that event \(A\) occurs given that event \(B\) occurs.

\[ P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0 \]

Intuition: Restrict the sample space to \(B\); ask what fraction of those outcomes also fall in \(A\).

Multiplication Rule

Relates intersections to conditional probabilities.

\[\begin{align} P(A \mid B) &= \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0 && \text{definition of conditional probability} \\[24pt] P(A \mid B)\,P(B) &= \frac{P(A \cap B)}{P(B)} \cdot P(B) && \text{multiply both sides by $P(B)$} \\[24pt] P(A \mid B)\,P(B) &= P(A \cap B) && \text{simplify} \\[24pt] P(B \mid A)\,P(A) &= P(A \cap B) && \text{swap $A$ and $B$} \\[24pt] P(A \cap B) &= P(A \mid B)P(B) = P(B \mid A)P(A) && \text{final multiplication rule} \end{align}\]

Intuition: To find the chance that both happen, compute the chance that one happens, then multiply by the chance the other happens given that.

Example problem

Remember this?

Out of 100 students:
- 40 like pizza
- 30 like burgers
- 10 of those included above like both pizza and burgers

Experiment: I select one student at random

Let:
- \(A =\) “student likes pizza”
- \(B =\) “student likes burgers”

  1. What is \(P(A \mid B)\), the probability that a student likes pizza given that they like burgers?
Answer

We start with the definition of conditional probability:

\[ P(A \mid B) = \frac{P(A \cap B)}{P(B)}. \]

Now substitute the known values:

\[ P(A \mid B) = \frac{0.10}{0.30}. \]

Simplify:

\[ P(A \mid B) = \frac{1}{3} \]

  1. What is \(P(B \mid A)\), the probability that a student likes burgers given that they like pizza?
Answer

We start with the definition of conditional probability:

\[ P(B \mid A) = \frac{P(A \cap B)}{P(A)}. \]

Now substitute the known values:

\[ P(B \mid A) = \frac{0.10}{0.40}. \]

Simplify:

\[ P(B \mid A) = \frac{1}{4} \]

  1. Are \(A\) and \(B\) independent? (Check whether \(P(A \cap B) = P(A)P(B)\) holds.)
Answer

To check independence, we compare \(P(A \cap B)\) with \(P(A)P(B)\).

First compute:

\[ P(A)P(B) = 0.40 \times 0.30 = 0.12. \]

Now compare with the actual value:

\[ P(A \cap B) = 0.10 \neq 0.12. \]

Therefore, \(A\) and \(B\) are not independent.

We can also see this because:

\[ P(A \mid B) = \tfrac{1}{3} \neq P(A) = 0.40. \]

  1. Compute \(P(A \cap B)\) (the probability a student likes both pizza and burgers).
Answer

We can use the multiplication rule to compute \(P(A \cap B)\) in two different ways:

  • Using \(P(A \cap B) = P(A \mid B)P(B)\)
  • Using \(P(A \cap B) = P(B \mid A)P(A)\)

Using \(P(A \mid B)P(B)\):

\[ P(A \cap B) = \left(\tfrac{1}{3}\right)(0.30) = 0.10. \]

Using \(P(B \mid A)P(A)\):

\[ P(A \cap B) = (0.25)(0.40) = 0.10. \]

Both methods agree with the given value:

\[ P(A \cap B) = 0.10. \]

  1. Suppose you learn that the student selected does not like burgers. What is \(P(A \mid B^c)\)?
Answer

We start with the definition of conditional probability:

\[ P(A \mid B^c) = \frac{P(A \cap B^c)}{P(B^c)}. \]

First compute the numerator (draw a venn diagram if this does not make sense):

\[ P(A \cap B^c) = P(A) - P(A \cap B) = 0.40 - 0.10 = 0.30. \]

Now compute the denominator:

\[ P(B^c) = 1 - P(B) = 1 - 0.30 = 0.70. \]

Substitute into the formula:

\[ P(A \mid B^c) = \frac{0.30}{0.70}. \]

Simplify:

\[ P(A \mid B^c) = \tfrac{3}{7} \approx 0.429. \]

  1. (Stretch) If I select two students at random with replacement, what is the probability that both like pizza?
Answer

Since the selections are with replacement, the draws are independent.

\[ P(\text{both like pizza}) = P(A) \times P(A). \]

Substitute values:

\[ P(\text{both like pizza}) = (0.40)(0.40). \]

Simplify:

\[ P(\text{both like pizza}) = 0.16. \]

  1. (Stretch) If I select two students at random without replacement, what is the probability that both like pizza?
Answer

Without replacement, the draws are dependent.

First student:

\[ P(\text{first likes pizza}) = \tfrac{40}{100}. \]

Second student (after one pizza-lover is removed):

\[ P(\text{second likes pizza} \mid \text{first likes pizza}) = \tfrac{39}{99}. \]

Multiply:

\[ P(\text{both like pizza}) = \tfrac{40}{100} \cdot \tfrac{39}{99}. \]

Simplify:

\[ P(\text{both like pizza}) = \tfrac{1560}{9900} = \tfrac{26}{165} \approx 0.158. \]

This is slightly less than the probability with replacement.

Tree Diagrams

We’ll model the sample space with a probability tree. Each edge is labeled by a (conditional) probability; each leaf is the product along its path and equals a joint probability.

Setup (from Problem 1):
- Out of 100 students: 40 like pizza, 30 like burgers, 10 like both.
- Let \(A\) = “likes pizza”, \(B\) = “likes burgers”.

From the counts:

\[ P(A)=0.40,\quad P(B)=0.30,\quad P(A\cap B)=0.10. \] Useful conditionals:

\[ P(B\mid A)=\frac{0.10}{0.40}=\frac{1}{4}=0.25,\qquad P(B\mid A^c)=\frac{P(B)-P(A\cap B)}{P(A^c)}=\frac{0.20}{0.60}=\frac{1}{3}. \] (And similarly \(P(A\mid B)=\tfrac{1}{3}\), \(P(A\mid B^c)=\tfrac{3}{7}\).)

Tree 1: Branch on \(A\) first, then \(B\)

  • Leaves give \(P(A\cap B)\), \(P(A\cap B^c)\), \(P(A^c\cap B)\), \(P(A^c\cap B^c)\).
Start
├── A (0.40)
│   ├── B       | P(B|A)=0.25  ⇒  P(A∩B)     = 0.40·0.25 = 0.10
│   └── B^c     | P(B^c|A)=0.75          ⇒  P(A∩B^c)   = 0.40·0.75 = 0.30
└── A^c (0.60)
    ├── B       | P(B|A^c)=1/3 ⇒  P(A^c∩B)   = 0.60·(1/3) = 0.20
    └── B^c     | P(B^c|A^c)=2/3          ⇒  P(A^c∩B^c) = 0.60·(2/3) = 0.40

Checks: \[ \begin{aligned} &\text{Row sums: } 0.10+0.30=0.40=P(A),\quad 0.20+0.40=0.60=P(A^c).\\ &\text{Column sums: } 0.10+0.20=0.30=P(B),\quad 0.30+0.40=0.70=P(B^c). \end{aligned} \]

Tree 2: Branch on \(B\) first, then \(A\)

  • Same joint probabilities appear; this is handy when a question conditions on \(B\).
Start
├── B (0.30)
│   ├── A       | P(A|B)=1/3   ⇒  P(A∩B)     = 0.30·(1/3) = 0.10
│   └── A^c     | 2/3          ⇒  P(A^c∩B)   = 0.30·(2/3) = 0.20
└── B^c (0.70)
    ├── A       | P(A|B^c)=3/7 ⇒  P(A∩B^c)   = 0.70·(3/7) = 0.30
    └── A^c     | 4/7          ⇒  P(A^c∩B^c) = 0.70·(4/7) = 0.40

How to Use the Tree

  • Joint probability of a leaf = product along its path.
    Example: \(P(A\cap B)=0.40\cdot 0.25=0.10\).
  • Conditional probability = branch label.
    Example: From the first tree, \(P(B\mid A)=0.25\).
  • Totals: add leaves in the relevant region.
    Example: “Exactly one likes” \(=P(A\cap B^c)+P(A^c\cap B)=0.30+0.20=0.50\).

Quick Practice (answers from the trees)

  • \(P(A\mid B)=\tfrac{1}{3}\), \(P(B\mid A)=\tfrac{1}{4}\)
  • \(P(A\cap B^c)=0.30\), \(P(A^c\cap B)=0.20\)
  • Exactly one likes pizza or burgers \(=0.50\)
  • \(P(A\mid B^c)=\tfrac{3}{7}\)

Tip: Draw whichever tree matches the given condition first (branch on the given event); it makes reading the needed conditional straight from the edge label.
