Key Terms

ğŸ“ What is the general form of a linear function?

A linear function can be written as:

\[ \hat{y} = mx + b \]

ğŸ”¢ What does the variable x represent?

The variable \(x\) represents the input to the function.

  • Itâ€™s the independent variable
  • You choose or observe values of \(x\)
ğŸ“ˆ What does the parameter m represent?

The parameter \(m\) represents the slope of the line.

  • It tells you how much the output \(\hat{y}\) changes for each 1-unit increase in \(x\)
  • A larger \(m\) makes the line steeper
  • If \(m > 0\), the line increases
  • If \(m < 0\), the line decreases
ğŸ“ What does the parameter b represent?

The parameter \(b\) is the y-intercept.

  • Itâ€™s the value of the function when \(x = 0\)
  • It shifts the line up or down on the graph
  • It does not affect the slope
ğŸ¯ What does the variable \(\hat{y}\) represent?

The variable \(\hat{y}\) represents the predicted output of the function.

  • Itâ€™s the dependent variable
  • It shows the modelâ€™s best estimate based on the input \(x\)

library(shiny)
library(ggplot2)

ui <- fluidPage(
  titlePanel("Linear Function"),
  
  # Plot output
  plotOutput("linePlot", height = "400px"),
  
  # Numeric inputs below the plot, side-by-side
  fluidRow(
    column(6,
           numericInput("m", "Slope (m):", value = 3, min = -5, max = 5, step = 0.1)
    ),
    column(6,
           numericInput("b", "Intercept (b):", value = 10, min = -20, max = 20, step = 1)
    )
  )
)

server <- function(input, output, session) {
  output$linePlot <- renderPlot({
    x <- seq(-10, 10, length.out = 100)
    y <- input$m * x + input$b
    df <- data.frame(x = x, y = y)
    
    ggplot(df, aes(x, y)) +
      geom_line(color = "#0072B2", linewidth = 1.5) +
      labs(
        title = bquote(hat(y) == .(input$m) * x + .(input$b)),
        x = "x", y = expression(hat(y))
      ) + 
      lims(x = c(0,10), y = c(0,40)) +
      theme_minimal(base_size = 16)
  })
}

shinyApp(ui, server)
