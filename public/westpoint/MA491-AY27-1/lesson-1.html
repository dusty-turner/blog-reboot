Lesson 1: Unboxing — Raspberry Pi & What Is a Computer

Objectives

NoteBy the end of this lesson, you will be able to:
  • Identify the major components of a Raspberry Pi (CPU, RAM, storage, GPIO, USB, HDMI, power)
  • Explain what makes something a computer and compare the Pi to a laptop or phone
  • Assemble a Raspberry Pi from kit components
  • Boot to Raspberry Pi OS and navigate the desktop
  • Use the file manager, terminal, and system settings
  • Connect the Pi to the internet

Before Class

  • No prep required. Kits will be distributed in class.
  • If you want to read ahead, skim the Raspberry Pi 5 specs page.

Lesson Outline

1. What Are We Doing This Semester? (5 min)

This is a data science course built on real infrastructure. You are not getting a pre-configured laptop with everything installed. You are going to build your own computing environment from scratch on a Raspberry Pi, learn to manage it like a professional, and use AI as a collaborator throughout the process.

By the end of the course you will have a working data science platform that you built, configured, and understand top to bottom.

2. Unbox the Kit (10 min)

Open your kit and identify each component. Lay them out on your desk.

Component What It Does
Raspberry Pi 5 board The computer itself — about the size of a credit card
MicroSD card Your hard drive (persistent storage)
USB-C power supply Provides power to the board
Case Protects the board (optional but recommended)
Keyboard, mouse, monitor Temporary peripherals — we go headless in Lesson 2
HDMI cable Connects the Pi to a monitor
Ethernet cable (if provided) Wired network connection
TipAI Assist

Not sure what a component is? Take a photo and ask your AI assistant to identify it and explain what it does. This is a great habit — use AI to fill gaps in real time.

3. What Is a Computer? (10 min)

Discussion: what makes something a computer? Your phone, your laptop, a server in a data center, this Raspberry Pi — they all have the same four things:

  1. Processor (CPU) — executes instructions
  2. Memory (RAM) — temporary workspace for running programs (lost on power off)
  3. Storage (MicroSD / SSD / hard drive) — persistent data that survives a reboot
  4. Input/Output — keyboard, monitor, network, USB, GPIO pins

Your Pi has a quad-core ARM CPU, 4-8 GB of RAM, and a MicroSD card for storage. A DoD supercomputer has the same building blocks — just more of them.

How is the Pi different from your laptop?

  • No built-in screen, keyboard, or battery
  • GPIO pins for connecting sensors and hardware (your laptop doesn’t have these)
  • Runs Linux instead of Windows or macOS
  • Costs $60 instead of $1,500
  • Same fundamental architecture

4. Assemble the Pi (15 min)

Step by step:

  1. Insert the MicroSD card into the slot on the underside of the board (pre-flashed with Raspberry Pi OS)
  2. Snap the board into the case if you have one
  3. Connect the HDMI cable from the Pi to your monitor
  4. Connect the keyboard and mouse via USB
  5. Connect Ethernet if available (more reliable than Wi-Fi for setup)
  6. Plug in the USB-C power supply — this is the ON switch. There is no power button on older models; the Pi 5 has a small power button.
WarningMicroSD Orientation

The MicroSD card goes in label-side up on the Pi 5. Don’t force it. If it’s not sliding in easily, flip it.

5. First Boot (10 min)

Once you plug in power, the Pi boots automatically. First boot takes 2-3 minutes as the OS configures itself.

You should see the Raspberry Pi OS desktop appear on your monitor. If you see a setup wizard, follow the prompts:

  • Set your locale and timezone
  • Connect to Wi-Fi (use the network credentials provided in class)
  • Set your password — write it down. If you forget it, you re-flash the card and start over.
  • Apply any system updates if prompted

6. Explore the Desktop (10 min)

Now that you have a running system, poke around:

  • File Manager — click the folder icon in the taskbar. This is your home directory (/home/pi/ or similar). Notice it looks like Windows Explorer or Finder, but the paths use forward slashes.
  • Terminal — click the black terminal icon in the taskbar (or right-click desktop and select “Open Terminal”). Type pwd and press Enter. This prints your current location in the file system. Type ls to list files.
  • Settings — click the Raspberry Pi menu (top-left) and look at Preferences. Check your screen resolution, Wi-Fi connection, and Bluetooth status.
# Try these in the terminal
pwd          # Where am I?
ls           # What files are here?
ls -la       # Detailed view (including hidden files)
hostname -I  # What is my IP address on the network?
TipAI Assist

If a command gives you output you don’t understand, paste it into your AI assistant and ask for an explanation. That’s what it’s there for.

7. Connect to the Internet (10 min)

If you used Ethernet, you’re probably already connected. Verify:

ping -c 3 google.com

If you see responses, you’re online. If not, check Wi-Fi:

# List available networks
sudo nmcli device wifi list

# Connect to a network
sudo nmcli device wifi connect "NETWORK_NAME" password "PASSWORD"

Once connected, update your system:

sudo apt update && sudo apt upgrade -y

This downloads and installs the latest software updates. It may take a few minutes.

8. Wrap-Up and Troubleshooting (5 min)

Quick check — raise your hand if any of these are NOT true:

  • Your Pi is assembled and powered on
  • You can see the desktop
  • You opened a terminal and ran pwd
  • You are connected to the internet (ping google.com works)

If something isn’t working, pair up with a neighbor or ask your AI assistant for troubleshooting steps. Common issues: MicroSD not fully inserted, wrong Wi-Fi password, monitor not on the right HDMI input.
