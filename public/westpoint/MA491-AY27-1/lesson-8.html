Lesson 8: ggplot2 – Grammar of Graphics

Overview

A good chart is worth a thousand print() statements. ggplot2 is the most widely used visualization package in R, and it works by layering components: you start with data, map variables to aesthetics, and add geometric objects. Once you internalize this grammar, you can build almost any chart you can imagine.

This lesson covers the core geoms, aesthetic mappings, faceting, themes, and labels. By the end, you should be able to look at a dataset and know which chart to reach for – and how to build it.

Objectives

NoteBy the end of this lesson, you will be able to:
  • Explain the grammar of graphics: data + aesthetics + geoms
  • Build plots with ggplot() + aes() + geom layers
  • Use common geoms: point, histogram, boxplot, bar, line, col
  • Map variables to color, fill, size, and shape
  • Split plots with facet_wrap() and facet_grid()
  • Apply themes and customize labels
  • Save plots with ggsave()
  • Choose the right chart type for a given question

The Grammar of Graphics

Every ggplot has three required components:

  1. Data – the data frame
  2. Aesthetics (aes()) – which variables map to which visual properties (x, y, color, size, shape)
  3. Geoms – the geometric objects that represent the data (points, bars, lines, etc.)
library(tidyverse)

# The minimal ggplot
ggplot(data = mtcars, aes(x = wt, y = mpg)) +
  geom_point()

Read it as: “Take mtcars, put wt on x and mpg on y, draw points.”

Everything else – colors, facets, themes, labels – is layered on top.

Common Geoms

Scatterplot: geom_point()

Best for: relationship between two continuous variables.

ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point(size = 3)

# Add a trend line
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point() +
  geom_smooth(method = "lm", se = TRUE)

Histogram: geom_histogram()

Best for: distribution of a single continuous variable.

ggplot(mtcars, aes(x = mpg)) +
  geom_histogram(bins = 12, fill = "steelblue", color = "white")
TipBins matter

Too few bins hide structure. Too many bins create noise. Start with the default, then adjust. Try binwidth as an alternative to bins.

Boxplot: geom_boxplot()

Best for: comparing distributions across groups.

ggplot(mtcars, aes(x = factor(cyl), y = mpg)) +
  geom_boxplot(fill = "lightblue") +
  labs(x = "Cylinders", y = "MPG")

Bar Chart: geom_bar() and geom_col()

geom_bar() counts rows for you. geom_col() uses a y variable you provide.

# geom_bar: counts automatically
ggplot(mtcars, aes(x = factor(cyl))) +
  geom_bar(fill = "darkgreen")

# geom_col: you supply the heights
mtcars |>
  group_by(cyl) |>
  summarize(avg_mpg = mean(mpg)) |>
  ggplot(aes(x = factor(cyl), y = avg_mpg)) +
  geom_col(fill = "darkgreen")

Line Chart: geom_line()

Best for: trends over time or ordered sequences.

ggplot(economics, aes(x = date, y = unemploy)) +
  geom_line(color = "firebrick") +
  labs(title = "US Unemployment Over Time",
       x = "Date", y = "Unemployed (thousands)")

Density Plot: geom_density()

Best for: smoothed distributions, especially when comparing groups.

ggplot(mtcars, aes(x = mpg, fill = factor(cyl))) +
  geom_density(alpha = 0.5) +
  labs(fill = "Cylinders")

Aesthetic Mappings

Aesthetics map data values to visual properties. Put them inside aes() when driven by data; put them outside aes() when setting a fixed value.

# Color mapped to a variable (inside aes)
ggplot(mtcars, aes(x = wt, y = mpg, color = factor(cyl))) +
  geom_point(size = 3)

# Color set to a fixed value (outside aes)
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point(size = 3, color = "steelblue")

# Multiple aesthetics
ggplot(mtcars, aes(x = wt, y = mpg, color = factor(cyl), size = hp)) +
  geom_point(alpha = 0.7)

Quick Reference

Aesthetic Geom types Example
color Points, lines Outline or line color
fill Bars, boxes, areas Interior color
size Points Point diameter
shape Points Point symbol (max 6 distinct shapes)
alpha All Transparency (0 = invisible, 1 = opaque)
linetype Lines Solid, dashed, dotted
Warningcolor vs. fill

color controls outlines and lines. fill controls interiors. A bar chart needs fill; a scatterplot needs color. Mixing them up is one of the most common ggplot mistakes.

Faceting

Split one plot into multiple panels by a categorical variable.

# facet_wrap: panels in a ribbon
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point() +
  facet_wrap(~cyl) +
  theme_minimal()

# facet_grid: rows and columns
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point() +
  facet_grid(gear ~ cyl) +
  theme_minimal()

# Free scales when ranges differ
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point() +
  facet_wrap(~cyl, scales = "free_y")
TipWhen to facet

Faceting is better than cramming multiple colors into one plot when you have 4+ groups or when the groups overlap heavily. Let each group breathe in its own panel.

Themes

Themes control the non-data appearance: background, gridlines, fonts.

p <- ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point()

# Built-in themes
p + theme_minimal()        # clean, light
p + theme_bw()             # black-and-white borders
p + theme_classic()        # no gridlines
p + theme_void()           # nothing but data

# Customize elements
p + theme_minimal() +
  theme(
    plot.title = element_text(face = "bold", size = 16),
    axis.title = element_text(size = 12),
    legend.position = "bottom"
  )

Labels

Always label your plots. Unlabeled axes are the hallmark of unfinished work.

ggplot(mtcars, aes(x = wt, y = mpg, color = factor(cyl))) +
  geom_point(size = 3) +
  labs(
    title = "Fuel Efficiency vs. Weight",
    subtitle = "Heavier cars get worse mileage",
    x = "Weight (1000 lbs)",
    y = "Miles per Gallon",
    color = "Cylinders",
    caption = "Source: mtcars dataset"
  ) +
  theme_minimal()

Saving Plots with ggsave()

# Save the last plot
ggsave("my_plot.png", width = 8, height = 5, dpi = 300)

# Save a specific plot object
my_plot <- ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point()
ggsave("my_plot.pdf", plot = my_plot, width = 8, height = 5)
TipFile formats

Use .png for presentations and web. Use .pdf for print and Quarto reports. Use .svg if you need to edit the graphic later.

Which Chart Should I Use?

Question Chart type Geom
How is one variable distributed? Histogram or density geom_histogram, geom_density
How do distributions compare across groups? Boxplot or violin geom_boxplot, geom_violin
What’s the relationship between two continuous variables? Scatterplot geom_point
How does a quantity change over time? Line chart geom_line
How do counts compare across categories? Bar chart geom_bar, geom_col
How do two categorical variables relate? Heatmap or stacked bar geom_tile, geom_bar(position = "fill")
WarningChart crimes

Avoid 3D charts, pie charts (use bar charts instead), dual y-axes, and truncated axes that exaggerate differences. Keep it simple and honest.

Putting It Together

# A polished, publication-ready chart
mtcars |>
  mutate(cyl = factor(cyl)) |>
  ggplot(aes(x = wt, y = mpg, color = cyl)) +
  geom_point(size = 3, alpha = 0.8) +
  geom_smooth(method = "lm", se = FALSE, linetype = "dashed") +
  facet_wrap(~cyl) +
  labs(
    title = "Weight vs. MPG by Cylinder Count",
    x = "Weight (1000 lbs)",
    y = "Miles per Gallon",
    color = "Cylinders"
  ) +
  theme_minimal() +
  theme(legend.position = "none")   # redundant with facet labels
